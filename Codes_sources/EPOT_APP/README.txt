Pour ouvrir les fichiers de l'application il faut installer l'ide android Studio avec le sdk android 9.0 revision 28 

Il se peut qu'un fichier soit rempli d'erreur, pour palier à ce souci il suffit de copier le contenu du fichier puis le supprimer et le recrée avec le même nom et la même extension (kotlin class file->class)
